# Graficadora+ 

Graficadora+ és una aplicació web interactiva per representar gràficament equacions i inequacions matemàtiques.

## Descripció

L'aplicació permet a l'usuari representar gràficament equacions i inequacions matemàtiques de manera molt visual, intuïtiva i atractiva. També permet visualitzar solucions de sistemes d’equacions i inequacions, mostrant clarament les interseccions i zones de solució.

## Característiques

- Gràfica interactiva amb coordenades cartesianes
- Zoom i panoràmica
- Modes clar/fosc
- Suport per equacions, inequacions i sistemes
- Teclat matemàtic integrat
- Exportació de gràfiques
- Interfície responsive
