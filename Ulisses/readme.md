Els viatges d'Ulisses: Un mapa interactiu
Aquesta és una aplicació web interactiva que visualitza el viatge èpic d'Ulisses (Odisseu) tal com es narra a l'Odissea d'Homer. El mapa mostra cada una de les catorze parades principals del seu viatge de tornada a casa, des de la caiguda de Troia fins a la seva arribada a Ítaca.

Característiques
Mapa interactiu: Fent servir la llibreria Leaflet.js, l'aplicació presenta un mapa navegable del mar Mediterrani.

Les 14 parades de l'Odissea: Cada punt del viatge està marcat amb un número que segueix l'ordre cronològic de la narració.

Informació detallada: En fer clic a cada marcador, s'obre una finestra emergent amb:

El nom del lloc.

Un breu resum del succés que hi va tenir lloc.

Una cita representativa del text d'Homer.

Una reflexió filosòfica que convida a pensar sobre els temes universals que planteja cada episodi (l'astúcia, la temptació, el lideratge, la nostàlgia, etc.).

Disseny clàssic: L'estètica de l'aplicació, amb tons ocres i fonts d'estil pergamí, busca evocar una sensació clàssica i literària.

Disseny adaptatiu (Responsive): La interfície s'adapta correctament a diferents mides de pantalla, des de dispositius mòbils a ordinadors d'escriptori.

Tecnologia
L'aplicació està construïda amb tecnologies web estàndard:

HTML5 per a l'estructura del contingut.

CSS3 per a l'estil, amb l'ajuda de:

Tailwind CSS per a un disseny ràpid i funcional.

Fonts de Google Fonts (Lora, Playfair Display, Cormorant Garamond).

JavaScript per a la interactivitat i la lògica del mapa.

Leaflet.js com a motor principal per a la creació de mapes interactius.
