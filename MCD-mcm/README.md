# MCD &middot; mcm - Guia Visual Interactiva

Benvingut/da a la guia visual interactiva per aprendre a calcular el **Màxim Comú Divisor (MCD)** i el **Mínim Comú Múltiple (mcm)**. Aquesta aplicació està dissenyada per facilitar la comprensió d'aquests conceptes matemàtics a través d'una interfície intuïtiva i pràctica.

## Característiques Principals

- **Descomposició Factorial Interactiva:** Descompon nombres en els seus factors primers de manera senzilla i visual.
- **Interfície d'Arrossegar i Deixar Anar (Drag-and-Drop):** Construeix el MCD i el mcm arrossegant els factors a les caselles corresponents.
- **Tres Modes d'Ús:** Tria entre un mode lliure, un mode de pràctica amb nivells de dificultat i un mode de resolució de problemes.
- **Feedback en Temps Real:** Rep correccions i pistes a l'instant per guiar el teu aprenentatge.
- **Disseny Net i Agradable:** Una interfície clara i sense distraccions per centrar-se en les matemàtiques.

## Modes de Joc

L'aplicació ofereix tres modalitats diferents per adaptar-se a les teves necessitats:

### 1. Mode Lliure

Ideal per experimentar i aprendre al teu ritme.

1.  **Introdueix els Nombres:** Escriu 2, 3 o 4 nombres enters positius als camps d'entrada.
2.  **Comença a Descompondre:** Fes clic a "Comença" per accedir a l'àrea de treball.
3.  **Arrossega els Factors Primers:** Utilitza la botonera de nombres primers per arrossegar els divisors a les columnes de descomposició de cada nombre.
4.  **Calcula el MCD i el mcm:** Un cop completada la descomposició, arrossega els factors a les files de "MCD" i "mcm" seguint les regles corresponents.
5.  **Comprova el Resultat:** Fes clic a "Comprova i calcula" per verificar si la teva solució és correcta.

### 2. Mode Pràctica

Posa a prova els teus coneixements amb exercicis de dificultat progressiva.

1.  **Tria la Dificultat:** Selecciona un nivell (Fàcil, Mitjana o Difícil).
2.  **Nou Conjunt:** Fes clic a "Nou conjunt" per generar un grup de nombres aleatoris.
3.  **Resol:** Segueix els mateixos passos que al mode lliure per descompondre els nombres i calcular el MCD i el mcm.

### 3. Mode Problemes

Aplica els teus coneixements a situacions del món real.

1.  **Nou Problema:** L'aplicació et presentarà un problema contextualitzat.
2.  **Analitza l'Enunciat:** Llegeix el problema, identifica els nombres clau i decideix si has de calcular el MCD o el mcm.
3.  **Introdueix les Dades:** Escriu els nombres i selecciona l'operació (MCD o mcm).
4.  **Resol:** Si les teves dades són correctes, s'activarà l'àrea de treball per a que resolguis el problema com en els altres modes.
