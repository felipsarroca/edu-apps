Practica el mètode per inspecció: ajusta els coeficients, consulta la pista quan la necessitis i compara les dues bandes de l'equació.

Taula d'àtoms

Element Reactius Productes Diferència

Guia ràpida

  1. Observa quins elements no estan equilibrats. Comença pels que apareixen en menys compostos.
  2. Ajusta els coeficients amb les fletxes o escrivint el nombre, mai modifiquis els subíndexs.
  3. Empra la taula d'àtoms per valorar la diferència entre bandes.
  4. Demana una pista si et bloqueges o mostra la solució per validar el procés complet.

Nivell actual

Mode per inspecció (nivell 1). Properes iteracions poden afegir el mètode algebraic i redox pas a pas.
